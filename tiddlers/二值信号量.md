## 🚦 二、类型 ①：二值信号量（Binary Semaphore）

### 🧩 概念

它的取值只有两种状态：
 **0（没信号）** 或 **1（有信号）**。

就像一个“门铃”或者“标志牌”，用来告诉别人“某件事发生了”。

### 🏠 类比：门铃

想象一下门铃系统：

- 当有人按门铃 → `Give()`：表示“事件发生了”；
- 里面的人听到铃声 → `Take()`：事件被接收；
- 如果没按铃，`Take()`的人就一直等（阻塞）。

📘 **使用场景：**

- 中断唤醒任务
- 任务之间同步（A 通知 B）

```c
xSemaphoreGiveFromISR(binarySem, NULL); // 中断触发事件
xSemaphoreTake(binarySem, portMAX_DELAY); // 任务等待事件
```

🧠 **一句话记忆：**

> 二值信号量 = “事件门铃”
>  谁按铃谁触发，谁听铃谁执行。
