## 中断向量表 (Interrupt Vector Table)

### 一句话概括

> 中断向量表就是一个**函数指针数组**，存放在 Flash 起始位置，告诉 CPU "发生某个中断/异常时，去哪个地址执行对应的处理函数"。

### 在内存中长什么样

以 STM32H7 为例，APP 起始地址 `0x08020000`：

```
地址            内容                    含义
─────────────────────────────────────────────────
0x08020000    0x24080000             ← 初始栈顶指针 (MSP)
0x08020004    Reset_Handler          ← 复位时从这里开始执行
0x08020008    NMI_Handler            ← 不可屏蔽中断
0x0802000C    HardFault_Handler      ← 硬件错误
0x08020010    MemManage_Handler      ← 内存管理错误
0x08020014    BusFault_Handler       ← 总线错误
0x08020018    UsageFault_Handler     ← 用法错误
0x0802001C    0 (保留)
...
0x0802002C    SVC_Handler            ← 系统调用
0x08020030    DebugMon_Handler
0x08020034    0 (保留)
0x08020038    PendSV_Handler         ← RTOS 上下文切换常用
0x0802003C    SysTick_Handler        ← 滴答定时器
─────────────────────────────────────────────────
0x08020040    WWDG_IRQHandler        ← 外设中断开始
0x08020044    PVD_IRQHandler
0x08020048    TAMP_STAMP_IRQHandler
...           ...                    ← 更多外设中断
```

### CPU 怎么用它

```
            ① 发生中断（比如 UART 收到数据）
                    │
            ② CPU 查 NVIC，得到中断号 N
                    │
            ③ 去向量表取地址：addr = *(SCB->VTOR + N*4)
                    │
            ④ 跳转到 addr 执行（即你写的中断处理函数）
                    │
            ⑤ 执行完返回，继续之前的代码
```

整个过程是**硬件自动完成**的，不需要软件查表。

### 和 Jump 函数的关系

CPU 上电复位时，硬件自动做两件事：

```
1. 从 0x00000000（或 VTOR 指向的地址）读第 0 项 → 设为 MSP
2. 从 偏移+4 读第 1 项 (Reset_Handler)        → 跳转执行
```

[main.c:110-120](vscode-webview://1g6e2e4h5pq3rdk2egrvuj6ot7gjd70lj8mapg40tb2tevrve3j6/main.c#L110-L120) 的 Jump 函数就是**用软件模拟了这个硬件复位过程**：

```c
__set_MSP(*(__IO UINT32*) appAddr);           // 模拟第1步：设栈指针
pApplicationFun = *(appAddr + 4);             // 模拟第2步：取复位向量
pApplicationFun();                            // 跳转执行
```

### VTOR 寄存器

默认情况下 CPU 从 `0x00000000` 找向量表。STM32 通过 `SCB->VTOR` 寄存器可以**重定位**向量表：

```c
// APP 启动后必须设置，否则中断还会跳到 Bootloader 的向量表
SCB->VTOR = 0x08020000;
```

这就是为什么前面提到 APP 如果忘记设置 VTOR，所有中断都会跑飞。

### 你在 STM32 项目中哪里定义的？

在启动文件 `startup_stm32h7xx.s` 中，用汇编定义：

```asm
__Vectors:
    DCD     __initial_sp              ; 栈顶
    DCD     Reset_Handler             ; 复位
    DCD     NMI_Handler               ; NMI
    DCD     HardFault_Handler         ; HardFault
    ...
```
