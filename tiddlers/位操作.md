 位操作符一览

| 操作符 | 名称     | 作用说明                                                     |
| ------ | -------- | ------------------------------------------------------------ |
| `&`    | 按位与   | 两个位都为 1 才得 1，否则为 0                                |
| `      | `        | 按位或                                                       |
| `^`    | 按位异或 | 两个位不同得 1，相同得 0                                     |
| `~`    | 按位取反 | 0 变 1，1 变 0（对整个数的所有位取反）                       |
| `<<`   | 左移     | 把所有位向左移动，右边补 0                                   |
| `>>`   | 右移     | 把所有位向右移动，左边补符号位（有符号数）或补 0（无符号数） |
